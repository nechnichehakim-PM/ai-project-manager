# Thèmes PM Hub

## Fichiers

- **pmhub-theme.css** — Thème actif (« Directeur de projet », sobre et classe).
- **theme-backup.css** — Sauvegarde de l’aspect d’origine (vert/teal).

## Revenir à l’ancien look

1. Dans chaque page HTML, remplacer  
   `href="themes/pmhub-theme.css"`  
   par  
   `href="themes/theme-backup.css"`  
   dans la balise `<link>` du thème.

2. Ou : copier le contenu de `theme-backup.css` dans `pmhub-theme.css` (et recharger les pages).

## Changer de police (optionnel)

Le thème « Directeur de projet » reste compatible avec les polices déjà chargées (Cormorant Garamond, DM Mono, Syne). Pour un rendu encore plus sobre, vous pouvez utiliser une police comme **Source Serif Pro** ou **Lora** pour les titres.
